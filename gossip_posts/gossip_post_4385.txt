旅客飆罵等不到行李 地勤人員下跪道歉 華航、桃勤回應了
: 2024-07-27 10:35 聯合報 記者周湘芸
: https://udn.com/news/amp/story/7266/8122299
: 凱米颱風影響，24日至25日不少國內外線航班停航，昨天起陸續恢復航班也造成各機場大
: 塞車，不少旅客光是領取行李就等待數小時，昨更有旅客因此現場飆罵地勤人員，使一名
: 地勤當場向旅客下跪道歉，引發熱議。中華航空今證實該名地勤為公司員工，並表示，感
: 謝所有一線同仁堅守崗位，盼大眾諒解值班人員於颱風期間執勤的辛勞。
: 凱米颱風挾帶豪大雨及強風，24日、25日共取消國內外959航班，昨天陸續恢復後，為盡
: 快協助滯留旅客疏運，不少業者紛紛加開班次服務旅客，也造成各機場量能緊繃。1名搭
: 乘外籍航空昨從韓國返台的陳姓旅客表示，光是在提領行李的地方就等了3小時，現場也
: 擠滿其他班機旅客，她抵達時，前面還有3班的行李要消化。
: 由於等候旅客過多，昨也有民眾認為等不到業者說明，忍不住飆罵地勤人員，更有一名華
: 航地勤當場向旅客下跪，畫面在網路瘋傳也引發網友熱議。
: 對此，華航表示，感謝所有一線同仁堅守崗位，即使颱風影響造成部分航機及行李延誤抵
: 達，仍竭力服務旅客，也盼大眾諒解值班人員於颱風期間執勤的辛勞，未能滿足所有旅客
: 的需求，尚祈見諒。
: 負責華航航班行李的桃勤公司也說，受凱米颱風遠離影響，各航班機於25日午間起陸續離
: 場，又於26日凌晨密集抵達，26日凌晨1至3點，桃勤提供服務進場客機即達22架次，較上
: 周五（同期）同時段超出近1倍。
: 桃勤公司指出，雖已就颱風影響航班動態提前部署作業人力，並安排現場人員加班完成作
: 業，但因夜間天候不佳影響作業速度，且同時段航班實過於密集抵達，瞬時作業量大增，
: 人力派遣有限度，導致行李運送作業延誤。
: 桃勤公司表示，此次受影響較大航班約有8班，對行李異常事件造成旅客久候及航空公司
: 不便深表歉意，後續將檢討改進特殊天候的人力調配，並依SOP及時妥適處理。
