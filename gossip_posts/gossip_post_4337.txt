挺無子女女性! 安妮斯頓怒嗆范斯：「望你女兒能順利生小孩!」范斯妹反擊!
張哲維綜合報導
近日，美國演員珍妮佛·安妮斯頓 ( Jennifer Aniston ) 批評了美國俄亥俄州聯邦參議
員范斯 ( J. D. Vance ) 針對無子女女性之言論，同時也公開支持賀錦麗 ( Kamala
Harris )。對此，范斯的妹妹琳賽 ( Lindsay ) 則發出聲明反駁。根據美國有線電視
新聞網《CNN》消息指出，近日，美國知名女演員珍妮佛·安妮斯頓 ( Jennifer
Aniston ) 頓於社交媒體上砲轟美國俄亥俄州聯邦參議員范斯 ( J. D. Vance )。令安
妮斯頓如此生氣的言論，可追溯至 2021 年。當時，范斯在新聞節目中，表示美國正被一
群沒有子女的女性所掌控，因為他們對於自己的生活感到不滿，接下來則想拖全美國人民
下水。其中，范斯更舉例了許多民主黨的內部人士，如現任美國副總統兼參議院議長卡馬
拉·哈里斯 ( Kamala Devi Harris )。而安妮斯頓對范斯過去關於無生育子女的女性
評論感到生氣。因此，她便在個人的 Instagram 限時動態中分享了范斯的影片，並寫道
 :「真不敢相信這句話居然出自有可能成為美國副總統的人之口。」
除此之外，她也諷刺的說道 :「祈禱范斯的女兒能幸運的生育自己的孩子，且不要求助試管嬰兒。」事實
上，安妮斯頓之所以如此生氣，也與她的個人背景有關。過去，她便時常被問及為何沒有
小孩。對此，她也數次公開坦承面臨到生育問題，以及曾嘗試試管嬰兒的經歷。因此，
近期安妮斯頓也表達了自身的政治立場，公開表示將支持賀錦麗 ( Kamala Harris )。
不過，針對安妮斯頓的批評，范斯的妹妹琳賽 ( Lindsay ) 亦發出聲明進行反駁。據悉
，在聲明中，她指出范斯其實是尊重女性的，因他的成長環境都受到女性的庇護。因此，
她認為民主黨相關人士的批評都屬惡意攻擊，全力為自己的弟弟辯護。
