標題: Re: [問卦] 「零日攻擊」怎那麼逼真？
時間: Fri Jul 26 00:20:14 2024

: https://www.youtube.com/watch?v=iAnZdVG041Y
: 看過預告後，我覺得這標題應該改為
: 「零日攻擊」怎那麼搞笑？
: 不說是曹興誠投資拍的，我還以為這是中共策劃找代理人拍的，用來恐嚇台灣 :)
你沒看顧問第一位是誰？沈柏楊阿
: 一個國家的戰爭紀律，與這國家的平時秩序是正相關的。
: 台灣社會的平時秩序，就決定在遇上緊急情況時是會有紀律而且服從性高。
: 若有台海危機或戰爭，台灣社會的表現會是像二戰時英國面對納粹圍攻時的表現。
: 預告中的社會表現，反而比較像是大陸社會的人心表現。:)
基本上失序會分成兩個階段
第一個階段是老共要打過來前可以參考96年飛彈危機，在不確定狀況下比較容易民心浮動
第二個階段會是在國軍遭受重大打擊，比如艦隊全滅，本島遭受大規模攻擊，甚至陸戰
當然這個階段最主要也是不確定
實際上過了這個階段反而民心會穩定
影片中失序速度太快，基本把警察等社會維穩系統當白痴
: 緊急情況下，台灣不一定能放人走。
一定會，這裡面甚至會帶走一大批有相當作戰能力的後備軍人
: 平時與戰時緊急處置是不同的，一旦國家動員並進入臨戰戒嚴狀態。
: 集體搶劫、集體強姦都是唯一死刑，速判速執行槍決，上面所謂的集體叛亂呢？
: 直接出動軍警重火力鎮壓，黑社會烏合之眾很快就鳥獸散了。
: 在戰時白目出來造反，算是自己找死，不能直接推翻政府，就要等著被政府殺光。
: 簡單說，既然都擺明是跟敵人站同一邊作戰，又不是正規軍，那就是該被處決的間諜。
影片中是鬼扯的，封鎖交流道？
事實上國軍和各警政單位，會在第一時間接管重要路口和後背的倉儲，大型空地比如
高速公路休息站等等
基本上像影片中一票腦殘的黑社會拿槍騎摩托車想去佔領交流道的下場就是被50打成肉渣
: 台灣政府的疫情表現，在世界上算是很優秀的，雖然有些中央官員的問題。
: 由此可見，台灣的政府體系運作是有效的，即使政務官可能有問題。
: 台灣面對中共威脅已經幾十年了，各種可能狀況應對都有應變處置方案，
: 作戰計劃會隨時局而不斷更動以適應新形勢，隨時都在準備作戰。
這個不一定因為我們的預案很多都是舊的，而都市更新的速度太快了
向新竹來說預定地直升機備用起降場是頭前溪公園，但是現在已經不能起降了
都市更新變成真正的公園了
: 這片像是文青幻想的戰情文藝片，想到以前那部國際橋牌社，都展露人文素養不夠深刻。
: 不知道曹興誠這些黑熊部隊到底研究了什麼？才會拍成這樣搞笑？
: 坦白說，看了影片預告的腦殘內容，反而擔心黑熊民兵在真正戰時會不會腦殘亂搞。
: 譬如在後方亂抓人去私刑吊路燈，反而影響民心安定，造成後方混亂。
: 畢竟曾經有些腦殘狂熱份子已有在說應該要抓人吊路燈了。
: 所以是不是一旦戰時動員，國軍就要馬上徵召黑熊民兵納入前線編制，
: 直接送去守第一線海防可以如願應對共軍，免得他們薰陶的腦殘民兵會在後方亂搞。
實際上最麻煩的地方是在於怎麼有效的動員平民真的為國作戰
從現實角度來說不可否認國軍對待義務役和志願役是有極大的區別的
義務役就是砲灰就算戰死也是砲灰價，之前國民黨好像是賴士葆吧就說過義務役更便宜
而相比義務役，志願役真的算是死得有價值，以最基本的入祀像烏坵海戰陣亡的義務役
士兵直到2012年才在馬英九的干預下得以入祀忠烈祠，而同時一起陣亡的14名高級志願
役軍官則早在1987年就已經入祀忠烈祠了
而已黑熊部隊的搞法真的會出現當年那個狀況開槍先打死這些亂搞的來波內戰然後就投
降了
不過顧問是沈柏楊帶頭會拍出這種東西其實也不意外
